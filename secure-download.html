Bug Log Last Updated: Aug 2025

Initial Command Error Being Fixed

The first command in the chat interface may appear to stall, freeze, or show a "SIGPIPE error". This is a known initialization issue that only affects the first command.

Solution

Stop the request and retry the same command. Subsequent commands will work normally.

Editor File Lock Being Fixed

Sometimes the file you are editing isn't captured by the editor and you get stuck editing only one file.

Solution

Close the chat box and reopen it on the new file you want to edit. The editor should then properly capture the selected file.

Setup Instructions

Windows Setup
▼
Step 1
Install WSL
PowerShell (Admin)
wsl --install
→ Reboot when prompted
→ Select Ubuntu when asked
→ Set up username and password
Step 2
Install Required Packages
WSL Terminal
sudo apt update && sudo apt install -y build-essential verilator autoconf gperf make gcc g++ bison flex
→ Installs Verilator and all required build tools
Step 3
Install Node.js (Optional)
WSL Terminal
sudo apt install -y nodejs npm
npm install -g @verilator/dev
→ Optional helper wrapper for Verilator
Step 4
Verify Installation
Any Terminal
wsl verilator --version
wsl make --version
wsl g++ --version
→ In VS Code: Open folder in WSL by clicking "Open in WSL"
→ Or enter WSL shell directly by typing 'wsl' in PowerShell/CMD
★
Add Convenient Aliases
WSL Terminal
alias v="verilator" alias m="make" alias cpp="g++"
source ~/.bashrc
→ Makes tools easier to run from PowerShell/CMD
→ Adds shortcuts: 'v' for verilator, 'm' for make, 'cpp' for g++
Step 5
Handle Windows Security Warning
Windows Security

⚠️ When you see this Windows security warning:

  1. Click the "More info" link (the blue underlined text)
  2. A new button "Run anyway" will appear
  3. Click "Run anyway" to install SigmanticAI
→ This security warning is normal for new applications not yet recognized by Microsoft
→ Following the steps above will safely install SigmanticAI
macOS Setup
▼
Step 1
Install Homebrew
Terminal
curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh | bash
Step 2
Install Verilator
Terminal
brew install verilator
Step 3
Verify Installation
Terminal
verilator --version
Step 4
Setup SigmanticAI Editor
Terminal
cd ~/Downloads unzip "SigmanticAI Editor.zip"
→ The DMG file may be in Downloads/Downloads folder
→ Double click the DMG to open it
→ Drag the SigmanticAI Editor into your Applications folder
cd ~
xattr -rd com.apple.quarantine "/Applications/SigmanticAI Editor.app"
→ This removes macOS security restrictions
